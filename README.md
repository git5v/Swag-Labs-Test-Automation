# 🧪 Swag Labs Test Automation

Automated testing suite for the [Sauce Demo](https://www.saucedemo.com/) web application.

## 🚀 Technologies Used
* **Language:** Python 3.x
* **Framework:** Pytest
* **Tool:** Selenium WebDriver
* **Reporting:** Allure Reports
* **Design Pattern:** Page Object Model (POM)

## 📦 Installation & Setup
1. **Clone the repo:**
   ```bash
   git clone [https://github.com/git5v/Swag-Labs-Test-Automation.git](https://github.com/git5ve/Swag-Labs-Test-Automation.git)

2. **Install dependencies:**
 pip install -r requirements.txt

3. **Note on Drivers:**
   Ensure your Chrome/Firefox version matches the drivers in your PATH
